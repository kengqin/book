# 《永恒道途》设定核查清单

---

## 使用说明

本文档用于创作过程中核查设定一致性，防止吃书。建议在以下时间点使用：
- **每章完成后**：核查本章设定是否与之前冲突
- **每卷完成后**：全面核查本卷设定
- **关键剧情前**：核查相关设定是否铺垫完整
- **大剧情转折**：核查是否符合设定逻辑

## 七、战力体系核查

### 战力平衡原则
1. **同阶无敌**：主角可越阶挑战，但不能跨越大境界
2. **大境界差距**：高一境界可轻松击败低一境界
3. **人数优势**：围攻可弥补部分境界差距
4. **特殊手段**：阵法、丹药、符箓可临时提升战力
5. **重伤状态**：重伤时战力大幅下降

### 主角战力核查表
| 境界 | 可战胜 | 可挑战 | 需逃跑 |
|------|--------|--------|--------|
| 炼气后期 | 炼气期 | 筑基初期 | 筑基中期+ |
| 筑基后期 | 筑基期 | 金丹初期 | 金丹中期+ |
| 金丹后期 | 金丹期 | 元婴初期 | 元婴中期+ |
| 元婴后期 | 元婴期 | 化神初期 | 化神中期+ |
| 化神后期 | 化神期 | 炼虚初期 | 炼虚中期+ |

**注意**：化神以上可越阶幅度逐渐减小！

### 天劫设定核查
- [ ] 小天劫：元婴、化神每300年一次
- [ ] 大天劫：炼虚、合体、大乘每3000年一次
- [ ] 飞升劫：大乘后期渡劫飞升
- [ ] 天劫威力随境界提升而增强
