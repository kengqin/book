# 小说书库阅读器 · Visual Studio

小说书库阅读器是“小说书库”桌面端的 Visual Studio 2022 配套扩展。它在开发环境中以只读方式展示本地书架、章节和正文，不修改解决方案、项目文件或源代码。

## 核心功能

- 在“小说书库”工具窗口中浏览书籍、章节和当前五行正文。
- 在代码编辑器中使用段落模式或紧凑的行尾模式显示内容。
- 固定显示当前章序、章名和全书总进度；章节切换可使用快捷键或阅读工具窗口。
- 逐行滚动至章节边界时自动切换相邻章节，并展示新章节的多行文字。
- 鼠标悬停在小说区域时滚动切行；移出后保持代码编辑器的正常滚动。
- 支持快捷键切行、切章、切换显示模式以及显示或隐藏代码内阅读。
- 通过本机 Bridge 与桌面端同步书架和阅读进度，重新打开 Visual Studio 后继续阅读。

## 系统要求

- Visual Studio 2022 Community、Professional 或 Enterprise
- 64 位 Windows 环境
- 已安装 Visual Studio Core Editor 组件
- 已安装并运行“小说书库”桌面端

## 开始使用

1. 在小说书库桌面端导入至少一本 TXT 或 EPUB 书籍。
2. 安装 VSIX，并按 Visual Studio Installer 提示完成安装。
3. 启动 Visual Studio，从“工具”菜单打开“小说书库”阅读器。
4. 在工具窗口中选择书籍与章节，再使用按钮、滚轮或快捷键控制阅读。

## 默认快捷键

| 功能 | Windows |
| --- | --- |
| 显示或隐藏代码内阅读 | `Ctrl+Alt+N` |
| 切换段落模式 / 行尾模式 | `Ctrl+Alt+9` |
| 上一行 / 下一行 | `Ctrl+Alt+↑` / `Ctrl+Alt+↓` |
| 上一章 / 下一章 | `Ctrl+Alt+←` / `Ctrl+Alt+→` |

在阅读工具窗口中点击“快捷键”可查看完整提示；点击“自定义快捷键”可进入 `工具 → 选项 → 环境 → 键盘`。用户自定义绑定始终优先。

## 数据与隐私

- 扩展不会直接访问或修改小说书库数据库。
- 小说正文仅作为只读的编辑器 Adornment 显示，不会写入项目文件。
- 所有书架读取与进度同步均通过回环地址上的本机 Bridge 完成。
- Bridge 使用桌面端运行时生成的临时端口和令牌，不向局域网或公网开放。

## 故障排查

- 阅读器无内容：确认小说书库桌面端正在运行，并重新打开工具窗口。
- 安装或卸载失败：完全关闭 Visual Studio 后，在桌面端插件页面重试。
- 快捷键冲突：打开 `工具 → 选项 → 环境 → 键盘`，搜索“小说书库”命令并重新绑定。
