# 小说书库阅读器 · JetBrains

小说书库阅读器是“小说书库”桌面端的 JetBrains IDE 配套插件，用于在开发环境中以只读方式浏览本地书架、章节和正文。插件不会修改项目文件，也不会将小说正文写入源代码。

## 核心功能

- 在“小说书库”工具窗口中选择书籍、章节并查看当前五行正文。
- 在代码编辑器中切换段落模式或紧凑的行尾模式。
- 固定展示当前章序、章名和全书总进度；章节切换可使用快捷键或阅读工具窗口。
- 滚动到章节边界时自动衔接相邻章节，一次展示新章节的多行内容。
- 鼠标悬停在阅读区域时滚动切行；移出后保留 IDE 原有的代码滚动行为。
- 通过快捷键切行、切章、切换显示模式和显示状态。
- 将书架、章节和阅读位置与桌面端同步，重新打开 IDE 后继续上次进度。

## 兼容范围

插件面向 IntelliJ Platform 2024.1 或更高版本，可用于 IntelliJ IDEA、PyCharm、WebStorm、Android Studio、Rider、CLion、GoLand、RubyMine 等兼容 IDE。

## 开始使用

1. 安装并启动“小说书库”桌面端，导入至少一本 TXT 或 EPUB 书籍。
2. 安装插件并重启 IDE。
3. 从右侧工具窗口栏打开“小说书库”，选择书籍与章节。
4. 使用工具窗口按钮、正文区域滚轮或快捷键控制阅读。

桌面端稍晚启动时，插件会自动重新连接，无需反复安装或手动重载项目。

## 默认快捷键

| 功能 | Windows / Linux |
| --- | --- |
| 显示或隐藏代码内阅读 | `Ctrl+Alt+N` |
| 切换段落模式 / 行尾模式 | `Ctrl+Alt+9` |
| 上一行 / 下一行 | `Ctrl+Alt+↑` / `Ctrl+Alt+↓` |
| 上一章 / 下一章 | `Ctrl+Alt+←` / `Ctrl+Alt+→` |

在工具窗口中点击“快捷键”可查看完整提示；点击“自定义快捷键”可直接进入当前 IDE 的 Keymap 设置。用户自定义绑定始终优先。

## 数据与隐私

- 插件不直接打开或修改 SQLite 数据库。
- 小说正文仅作为只读的编辑器 Inlay 显示，不写入源文件。
- 书架与进度仅通过回环地址上的本机 Bridge 交换。
- Bridge 使用桌面端运行时生成的临时端口和令牌，不向局域网或公网开放。

## 故障排查

- 工具窗口无内容：确认桌面端已运行，稍候数秒等待自动重连。
- 插件更新或卸载失败：保存工作并完全关闭正在使用插件文件的 JetBrains IDE 后重试。
- 快捷键冲突：打开 `Settings / Preferences → Keymap`，搜索“小说书库”并重新绑定。
