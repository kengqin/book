# 小说书库 VS Code / Cursor 阅读器

这个插件把桌面端书库中的小说以只读装饰方式显示在当前代码行末尾，不会修改或保存到源文件。

## 使用

1. 启动小说书库桌面端，并确认已经导入小说。
2. 安装插件后执行 `Developer: Reload Window`，或重启 VS Code / Cursor。
3. 点击左侧活动栏的“小说书库”图标。侧边栏按“书架、章节、正文”展示桌面端数据；点击书名或章节可直接切换。“滚轮阅读区”固定显示当前 5 行，鼠标悬浮在该区域滚动可切换上一行/下一行，移出后 VS Code 保持正常滚动。
4. 使用阅读器标题栏的图标显示或隐藏代码内阅读、切换段落/行尾显示模式、逐行滚动、切换章节和刷新书架。段落模式显示完整自然段落，行尾模式保留原有的紧凑行尾提示效果。也可以在任意编辑器中使用快捷键：
   - `Ctrl+Alt+N`：开启或关闭编辑器内阅读
   - `Ctrl+Alt+9`：切换段落模式 / 行尾模式
   - `Ctrl+Alt+Up/Down`：上一行 / 下一行
   - `Ctrl+Alt+Left/Right`：上一章 / 下一章
   - `Ctrl+Alt+D`：打开小说书库桌面端
5. 点击阅读器标题栏的键盘图标，可以查看默认快捷键并直接打开 VS Code / Cursor 的快捷键设置；用户绑定会覆盖插件默认值。

阅读器每次显示 5 行，切行和切章进度会回写桌面端书库，重新打开 IDE 后从同一章节和行位置继续。显示/隐藏状态会跨重启保存；隐藏时翻行或切章不会自动重新显示正文。
