# 小说书库桌面端使用说明

## 当前功能

- TXT 与 EPUB 文件选择、独立 Worker 后台解析
- UTF-8、GB18030、UTF-16 LE / BE 编码识别
- 本地书架与 SQLite 持久化
- 书籍详情、分卷目录和章节阅读
- EPUB2 / EPUB3 元数据、封面、嵌套目录、插图和富文本排版
- 字号、行距和阅读配色
- 当前章节和章内滚动进度恢复
- 书名、作者、章名和正文搜索
- 删除本地书籍
- JSON 书库备份与恢复
- 工具库与本地富文本笔记
- 笔记搜索、置顶、复制、删除和自动保存
- 单篇笔记 Markdown、HTML、JSON 导出与全部笔记 JSON 导入导出
- Windows NSIS 安装包
- 启动自动检查更新、应用内更新安装与重启
- 历史版本更新日志和指定版本安装入口
- 可配置关闭窗口时询问、缩小到系统托盘或直接退出

## 数据位置

桌面端数据保存在：

```text
%APPDATA%\NovelLibrary\library.db
```

应用不会使用 `com.kengqin.novellibrary` 或中文目录保存书库数据。

## 开发命令

```bash
npm run desktop:dev
npm run desktop:web:build
npm run desktop:build
```

`desktop:dev` 需要当前终端能够执行 `cargo --version`。

## Windows 产物

开发构建：

```text
apps/desktop/src-tauri/target/debug/novel-library-desktop.exe
```

Release 可执行文件：

```text
apps/desktop/src-tauri/target/release/novel-library-desktop.exe
```

NSIS 安装包：

```text
apps/desktop/src-tauri/target/release/bundle/nsis/NovelLibrary_0.3.2_x64-setup.exe
```

安装时按本机约定选择 `D:\Software\NovelLibrary`。安装产品标识固定为
`NovelLibrary`，不会再生成中文安装目录；应用窗口标题仍显示“小说书库”。
NSIS 安装向导包含安装位置选择页，软件安装目录可以由用户修改。

## 版本与更新

侧栏“版本”页面会显示当前版本、可用更新和所有历史版本的更新内容。默认在应用启动时静默检查稳定版本；发现新版本后可以直接下载、安装并重启。

更新下载进度会固定显示在应用右下角，切换页面不会中断。下载过程中可以使用右上角关闭按钮取消；下载完成后由用户点击“安装并重启”，应用不会在下载结束时强制重启。

历史版本通过对应 GitHub Release 安装包安装。降级前必须先在设置页导出书库备份，因为旧版本可能不支持更高版本的数据库 schema。

更新包使用 Tauri 签名校验。私钥保存在发布环境，不提交到仓库；仓库只保存更新公钥。

桌面端首次使用用户安装目录内的 `NovelLibraryData` 作为默认数据目录。更新后的首次启动会把旧版 `%APPDATA%\NovelLibrary` 中的受管书库和配置迁移到这里，成功后清理旧版受管文件。设置页选择其他目录时会复制当前书库、校验迁移结果、同步安装目录配置，并清理旧数据库文件。

“数据库文件”右侧的“更改”允许直接选择或新建 `.db`、`.sqlite`、`.sqlite3` 文件，文件名和所在目录都可以自定义。选择结果写入安装目录内的 `NovelLibrary.storage.json`，下次启动继续使用；旧版 C 盘配置只在迁移期间读取。

## 关闭窗口与托盘

默认点击窗口关闭按钮时会询问“缩小到托盘”还是“直接退出”。勾选“记住我的选择”后，下次关闭会直接执行上次选择。设置页的“关闭窗口时”下拉框可以随时切换为“每次询问”“缩小到托盘”或“直接退出”。

缩小到托盘不会退出应用，IDE 插件仍可同步书库和阅读进度。双击托盘图标或使用托盘菜单“显示主窗口”可以恢复窗口；直接退出后，需要重新打开小说书库才能继续同步。

关闭偏好保存在安装目录内的 `NovelLibraryData\app-settings.json`，不包含书库正文或阅读记录。

## 备份恢复

在桌面端“设置”页面中：

- “导出”会生成 `NovelLibrary-backup-YYYY-MM-DD.json`。
- “恢复”会合并备份中的书籍，并按书籍 ID 更新已有记录。
- 备份包含书籍、章节、主题、解析配置、阅读进度和本地笔记。

## 本地书籍导入

书架的“导入书籍”支持 `.txt` 和 `.epub`。TXT 继续使用编码识别、章节正则、广告清理和断行合并；EPUB 使用独立解析器读取 OPF、书脊、NAV / NCX、封面和图片，两种格式会进入同一套书架、搜索、备份和阅读进度系统。

EPUB 正文以安全结构化内容解析后再生成阅读 HTML，不执行电子书内的脚本、事件属性或第三方样式。图片转换为本地 data URL，移动数据库或恢复备份后仍可离线显示。

## 工具库与本地笔记

主导航点击“工具”进入工具库，再选择“本地笔记”进入独立笔记工作区。工具库只负责聚合入口，笔记列表和编辑器不会额外占用一条工具侧栏。

本地笔记使用当前 `library.db` 中的 `notes` 表保存，跟随数据目录和数据库文件一起迁移。编辑正文后自动保存；切换笔记前会等待当前保存完成。

- 列表顶部支持搜索、新建、全部 JSON 导入和全部 JSON 导出。
- 编辑区支持富文本格式、置顶、创建副本和删除。
- 单篇笔记可以导出为 Markdown、HTML 或 JSON。
- 完整数据备份会同时包含书库与笔记。
