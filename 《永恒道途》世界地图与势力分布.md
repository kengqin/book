# 《永恒道途》世界地图与势力分布

---

## 凡尘界：天玄大陆详细地图

### 整体地形
天玄大陆呈不规则圆形，直径约10万里，周围是无尽海域。

#### 五大区域

**1. 东荒（青云州位于此）**
- **面积**：占大陆25%
- **地形**：多山脉、森林，蛮荒之地
- **灵气浓度**：中等偏下
- **特点**：
  - 妖兽横行，人族势弱
  - 资源丰富但危险
  - 被称为"流放之地"
- **主要势力**：
  - **青云宗**（七品，正道，主角起始宗门）
  - **血魔宗**（七品，魔道）
  - **万妖谷**（七品，妖族）
  - **散修联盟**（中立）
  - **百宝阁**（商会，中立）

**2. 西漠**
- **面积**：占大陆20%
- **地形**：沙漠、戈壁、绿洲
- **灵气浓度**：中等
- **特点**：
  - 佛国圣地，寺庙林立
  - 民风淳朴，佛法昌盛
  - 妖兽较少，相对安全
- **主要势力**：
  - **大雷音寺**（八品，佛门领袖）
  - **悬空寺**（七品）
  - **白马寺**（七品）
  - **十八罗汉寺**（六品以下，众多）

**3. 南岭**
- **面积**：占大陆20%
- **地形**：丛林、沼泽、瘴气之地
- **灵气浓度**：高，但有毒
- **特点**：
  - 万毒之地，毒修聚集
  - 毒虫猛兽众多
  - 危险但机缘也多
- **主要势力**：
  - **万毒宗**（八品，毒道领袖）
  - **五毒教**（七品）
  - **蛊神宗**（七品）
  - **药王谷**（七品，以毒攻毒）

**4. 北原**
- **面积**：占大陆20%
- **地形**：冰雪覆盖，冰原、雪山
- **灵气浓度**：高，偏冰属性
- **特点**：
  - 冰雪世界，武道昌盛
  - 人族与雪族共存
  - 出产冰系灵药
- **主要势力**：
  - **雪族**（八品，北原霸主）
  - **冰魄宗**（七品）
  - **武神殿**（七品，武道领袖）
  - **寒山寺**（六品，佛门）

**5. 中州**
- **面积**：占大陆15%
- **地形**：平原、丘陵，最适合人类居住
- **灵气浓度**：最高
- **特点**：
  - 文明中心，最繁华之地
  - 宗门林立，强者如云
  - 九品宗门都在此
- **主要势力**：
  - **太虚宗**（九品，正道领袖）
  - **天魔宗**（九品，魔道领袖）
  - **万妖盟**（九品，妖族领袖）
  - **天剑宗**（九品，剑道领袖，叶无双所在）
  - **天机阁**（九品，情报）
  - **百宝阁总部**（九品，商会）

### 东荒详细地图（主角前期活动区域）

#### 地理分布
**从北到南**：
1. **黑风山脉**（最北）
   - 长度：3000里
   - 特点：妖兽众多，低阶修士试炼之地
   - 资源：低阶灵药、妖兽材料

2. **青云山脉**（中部）
   - 主峰：青云峰（高万丈）
   - 势力：青云宗
   - 资源：中等灵脉、灵药园

3. **血魔山**（东部）
   - 特点：终年血雾笼罩
   - 势力：血魔宗
   - 危险度：极高

4. **万妖林**（南部）
   - 面积：方圆万里
   - 特点：妖族栖息地
   - 势力：万妖谷

5. **迷雾沼泽**（西部）
   - 特点：毒雾弥漫，容易迷路
   - 资源：毒系灵药
   - 危险度：高

#### 主要城市与坊市
1. **青云城**
   - 位置：青云宗山脚下
   - 规模：百万人口
   - 特点：东荒最大城市，修仙者与凡人混居
   - 设施：百宝阁分店、散修联盟分部、各大店铺

2. **黑风城**
   - 位置：黑风山脉入口
   - 规模：10万人口
   - 特点：冒险者聚集地，混乱之地

3. **血色城**
   - 位置：血魔宗外围
   - 规模：5万人口
   - 特点：魔道修士聚集地，正道禁地

4. **万妖城**
   - 位置：万妖林边缘
   - 规模：妖族专属
   - 特点：人类禁入，除非获得许可

### 重要秘境与遗迹

#### 凡尘界级
1. **古剑冢**
   - 位置：东荒深处
   - 开启条件：每百年开启一次
   - 内容：上古剑修的墓地，可获得剑道传承
   - 危险度：高
   - 主角获得：完整版《九天剑典》

2. **丹王洞府**
   - 位置：南岭深处
   - 开启条件：需要丹王令
   - 内容：上古丹王的洞府，有丹道传承
   - 危险度：中
   - 主角获得：丹道传承、大量丹方

3. **龙墓**
   - 位置：东海海底
   - 开启条件：龙族血脉或龙族许可
   - 内容：上古龙族墓地
   - 危险度：极高
   - 主角获得：龙族神通、龙血淬体

4. **仙魔战场**
   - 位置：虚空之中，随机出现
   - 开启条件：随机
   - 内容：上古仙魔大战遗迹
   - 危险度：极高
   - 主角获得：远古功法、法宝

#### 区域级
1. **青云秘境**
   - 位置：青云宗内
   - 开启条件：宗门大比前十
   - 内容：青云宗历代祖师的传承
   - 危险度：中

2. **血魔窟**
   - 位置：血魔宗内
   - 开启条件：血魔宗核心弟子
   - 内容：血魔宗传承
   - 危险度：高

3. **万妖塔**
   - 位置：万妖谷
   - 开启条件：妖族天骄
   - 内容：妖族传承
   - 危险度：高

## 灵霄界：五大洲详细地图

### 整体结构
灵霄界比凡尘界大万倍，由五大洲和无尽海域组成。

#### 五大洲

**1. 东胜神洲（道门祖地）**
- **面积**：占灵霄界20%
- **灵气浓度**：极高
- **特点**：
  - 道门祖地，三清道统
  - 仙山福地最多
  - 灵气最浓郁
- **主要势力**：
  - **太清宗**（超级势力，人教道统）
  - **玉清宗**（超级势力，阐教道统）
  - **上清宗**（超级势力，截教道统）
  - **昆仑派**（超级势力）
  - **蜀山剑派**（超级势力，剑道圣地）
- **重要地点**：
  - **昆仑山**：万山之祖，道门圣地
  - **蓬莱仙岛**：海外仙山
  - **方丈山**：仙家福地
  - **瀛洲**：神秘之地

**2. 西牛贺洲（佛门圣地）**
- **面积**：占灵霄界20%
- **灵气浓度**：高，带佛性
- **特点**：
  - 佛门圣地，寺庙林立
  - 信仰之力浓厚
  - 妖族较少
- **主要势力**：
  - **大雷音寺**（佛门领袖）
  - **悬空寺**（超级势力）
  - **白马寺**（超级势力）
  - **少林寺**（超级势力）
- **重要地点**：
  - **灵山**：佛门圣地
  - **须弥山**：佛教圣山
  - **恒河**：佛门圣河

**3. 南瞻部洲（人族势力）**
- **面积**：占灵霄界25%
- **灵气浓度**：高
- **特点**：
  - 人族势力，百家争鸣
  - 国家林立，王朝更迭
  - 修仙与世俗结合最紧密
- **主要势力**：
  - **大周王朝**（人族王朝）
  - **大秦帝国**（人族帝国）
  - **儒道书院**（超级势力，儒道领袖）
  - **兵家**（超级势力）
  - **墨家**（超级势力）
- **重要地点**：
  - **长安**：大周首都
  - **咸阳**：大秦首都
  - **稷下学宫**：百家争鸣之地

**4. 北俱芦洲（妖族领地）**
- **面积**：占灵霄界25%
- **灵气浓度**：高，偏妖异
- **特点**：
  - 妖族领地，万妖之国
  - 人族禁地，危险重重
  - 妖兽天堂
- **主要势力**：
  - **妖族五帝**（青龙、白虎、朱雀、玄武、麒麟五族）
  - **万妖盟**（超级势力）
  - **天妖宫**（超级势力）
- **重要地点**：
  - **妖都**：妖族首都
  - **万妖林**：妖族圣地
  - **不死山**：凤凰族领地

**5. 中州神土（万族交汇）**
- **面积**：占灵霄界10%
- **灵气浓度**：最高
- **特点**：
  - 万族交汇，强者如云
  - 最繁华，也最危险
  - 天骄聚集地
- **主要势力**：
  - **万宗联盟**（各宗联合体）
  - **天机阁**（情报组织）
  - **百宝阁总部**（商业组织）
  - **杀手组织"暗影"**（暗杀组织）
- **重要地点**：
  - **万宗城**：万宗联盟总部
  - **天机城**：天机阁总部
  - **万宝城**：商业中心

### 无尽海域
- **面积**：比五大洲加起来还大
- **特点**：
  - 海中妖族无数
  - 海岛林立
  - 资源丰富但危险
- **主要势力**：
  - **龙宫**（海族霸主）
  - **四海龙族**（东海、西海、南海、北海）
  - **散修岛屿**（众多）
- **重要地点**：
  - **东海龙宫**：海族中心
  - **方丈岛**：三仙岛之一
  - **蓬莱岛**：三仙岛之一
  - **瀛洲岛**：三仙岛之一

### 虚空秘境
- **类型**：依附于灵霄界的小世界
- **数量**：成千上万
- **特点**：
  - 独立的空间
  - 有各种机缘
  - 有进无出是常态
- **著名秘境**：
  - **远古战场**：上古大战遗迹
  - **仙人洞府**：陨落仙人的洞府
  - **妖族圣地**：妖族先贤的墓地
  - **混沌秘境**：混沌之气弥漫

### 仙界主要势力

#### 天庭
- **统治者**：玉皇大帝
- **结构**：
  - 玉帝：最高统治者
  - 四御：辅佐玉帝
  - 五方五老：各方仙王
  - 六司：职能部门
  - 雷部、火部、斗部、财部等
- **军队**：天兵天将
- **特点**：名义上统治仙界，实则各方割据

#### 阐教
- **教主**：元始天尊
- **道场**：昆仑山
- **特点**：
  - 注重根脚，收徒严格
  - 天道正统
  - 与截教对立
- **主要人物**：
  - 十二金仙
  - 姜子牙
  - 哪吒（后期）

#### 截教
- **教主**：通天教主
- **道场**：碧游宫
- **特点**：
  - 有教无类，万仙来朝
  - 实力强大但良莠不齐
  - 与阐教对立
- **主要人物**：
  - 四大弟子
  - 赵公明
  - 三霄娘娘

#### 人教
- **教主**：太清老子
- **道场**：首阳山
- **特点**：
  - 人族道统
  - 弟子稀少但都是精英
  - 保持中立
- **主要人物**：
  - 玄都大法师

#### 西方教
- **教主**：接引道人、准提道人
- **道场**：灵山
- **特点**：
  - 后来发展为佛教
  - 渡化有缘
  - 经常挖其他教墙角

#### 散仙联盟
- **性质**：不加入任何势力的仙人联合体
- **特点**：
  - 实力强大但松散
  - 逍遥自在
  - 不受天庭约束

## 神界：洪荒大陆

### 整体结构
神界是一片无边无际的洪荒大陆，比仙界大千百倍。

#### 主要区域

**1. 中央神土**
- **统治者**：神庭
- **特点**：
  - 神界中心，灵气最浓郁
  - 神庭统治
  - 最强者聚集
- **主要势力**：
  - **神庭**（神界名义统治者）
  - **八大神族**（古神后裔）

**2. 东方苍木域**
- **特点**：森林覆盖，木属性灵气浓郁
- **主要势力**：木族、精灵族

**3. 南方炎火域**
- **特点**：火山众多，火属性灵气浓郁
- **主要势力**：火族、炎魔族

**4. 西方庚金域**
- **特点**：金属矿藏丰富，金属性灵气浓郁
- **主要势力**：金族、矮人族

**5. 北方玄水域**
- **特点**：海洋湖泊众多，水属性灵气浓郁
- **主要势力**：水族、海族

**6. 中央厚土域**
- **特点**：大地之力浓郁
- **主要势力**：土族、巨人族

#### 混沌海
- **位置**：洪荒大陆周围
- **特点**：
  - 混沌之气弥漫
  - 危险重重
  - 通往混沌界

## 传送体系

### 凡尘界
- **传送阵**：城市间传送
- **距离**：千里到万里
