# 在线小说书库

基于 VitePress 的多书阅读项目。每本书的内容与资料独立存放在 `书库/<书名>/`，平台主题和构建配置保留在 `正文/`。

## 目录结构

```text
书库/
├── 剑来/
│   ├── 原文/       # 本地原始留档，不提交 Git
│   └── 正文/       # 拆分后的发布章节
└── 永恒道途/
    ├── 正文/
    ├── 设定/
    ├── 参考资料/
    └── 站点/

正文/               # VitePress 平台配置与公共主题
scripts/            # 内容整理脚本
```

## 当前展示

当前站点展示《剑来》本地留档中已整理的章节。《永恒道途》已完整归档，后续可继续重构。
