# 第十六章 内门大比（上）

内门大比当日，青云宗演武场人潮如海。

与外门常规比斗不同，这场大比直接决定外门弟子的去留与层级。前十可入内门候选，前三可获长老面见资格。看台上不仅有内门弟子，连几位平日不露面的执事长老也到了。

陈玄持号牌站在丙字擂下，灰衣洗得发白，在一片锦袍间并不起眼。

“第一轮，陈玄，对刘嵩。”

刘嵩炼气七层，刀法稳重。开场后两人试探十招，陈玄以步法拉开角度，抓住刘嵩收刀时一息空窗，剑背击腕，逼其脱刀认输。

第一轮，胜。

第二轮对手是炼气七层巅峰体修，硬功极强。陈玄不与其正碰，绕擂游走，以连续剑气磨其护体灵光，最后趁对方灵力回落，一记“回云斩”斜破肩胛，再胜。

两战连捷后，他名字被挂进前六十四榜。

午后第三轮，压力陡增。

陈玄抽到的对手叫杜震，外门成名已久，炼气八层，擅雷系术法。杜震一上台便两手掐诀，雷弧如蛇般在擂面窜走，把陈玄活动空间压到半边角落。

“你前两场靠步法偷赢。”杜震冷声道，“这场你没地方跑。”

陈玄没有反驳，只把呼吸压到最稳。

雷法快，耗灵也快。杜震想速决，他就偏不让速决。前二十息里，陈玄连出七剑，都只为卸雷不为伤敌，手臂被震得发麻，虎口再裂，却硬是顶住了第一波雷暴。

看台上嘘声四起：“光挨打有什么用？”

陈玄听见了，却当没听见。

第二十一息，杜震雷诀出现半拍停顿。陈玄脚下一错，直切近身，青锋剑贴着雷弧缝隙挑向杜震肋下。杜震急转身，仍慢半步，衣袍被划开，鲜血渗出。

“找死！”

杜震怒喝，强行提灵施展《惊雷坠》。一道粗雷自上而下砸落，陈玄被雷芒擦中左肩，半边身子瞬间麻木，几乎跪地。

这一下若换别人，多半已输。

陈玄却借落雷激起的烟尘，咬牙逼出丹田剑气，连斩三道青芒。第一道逼杜震抬臂格挡，第二道切其脚下阵纹，第三道直取咽喉前三寸。

杜震瞳孔一缩，急停后仰，整个人跌出擂台边线。

“丙字擂，陈玄胜！”

全场先静后哗。

炼气七层逆胜炼气八层，这一战把原本“药园黑马”的称呼坐实。连看台高处的几位长老也多看了陈玄两眼。

陈玄下台时，左肩仍在发颤，内息也乱。他没有接受围观弟子的恭维，径直去医棚包扎，再回到候战区闭目调息。

因为赛程没有给任何人喘息太久。

傍晚第四轮，争前十六。

对手沈渊，炼气八层中期，枪修。此人枪势大开大合，起手便是连环十二刺，逼得陈玄节节后退。两人缠斗近百招，陈玄胸腹连中枪背，气血翻腾，几次险些出界。

关键时刻，他把青元剑诀第一层行气压到极限，硬顶一记枪尾横扫，贴枪而进，一剑点中沈渊握枪虎口。沈渊吃痛失衡，陈玄顺势转身回斩，将其木枪斩断半截。

兵器断，胜负分。

“陈玄，晋前十六！”

夜幕降临时，榜墙更新。

前十六名单里，陈玄二字被红墨重重点出。外门弟子院里一片议论，有人惊叹，有人嫉妒，也有人开始重新评估这个出身杂役的少年。

陈玄回屋后却只做了两件事：敷药、打坐。

今日四战，已把他灵力消耗近七成。若明日继续硬拼不调息，必败无疑。

他在昏暗烛火前缓缓运转周天，直到午夜才收功。起身时，他摸了摸心口处的掌天瓶，目光沉静。

前十六不是终点。

真正决定命运的战，还在明天。

（第十六章完）
